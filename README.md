RustでVimライクなテキストエディタをスクラッチで書いてみるプロジェクト

# できること
* 文書の表示
* 画面サイズを超えた文書に対するスクロール
* 文書の編集
	* INSERT, NORMALモードをスイッチする
* 基本的なNORMALモードコマンド
* 画面下に現在のモード・カーソル位置を表示する
* 画面表示の差分アップデート
	* ちらつき防止になる
	* スクロール時も画面表示をずらして差分アップデートする
